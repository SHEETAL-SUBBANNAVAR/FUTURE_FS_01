# 🚀 Pro Portfolio Website (Full Stack)

A modern, responsive **full-stack portfolio website** built with a clean UI, smooth animations, and a working backend contact system.

This project showcases my skills, projects, and provides a direct way for recruiters to contact me.

---

## 🔥 Features

* 🌙 Dark / Light mode toggle
* 📱 Fully responsive design
* ⚡ Smooth scroll animations
* 🎨 Modern glassmorphism UI
* 📩 Contact form with backend integration
* 🗄️ Stores messages in SQLite database
* 📧 Email notifications using Nodemailer
* 🔒 Rate limiting (anti-spam protection)

---

## 🛠️ Tech Stack

### Frontend

* HTML
* CSS (Custom UI + Animations)
* JavaScript

### Backend

* Node.js
* Express.js
* SQLite

### Tools

* Vite
* Nodemailer
* Lucide Icons
